## Branch name guide line

Branch type

- feat (for feature branches)
- chore (for tasks, general refactors, adding env variables etc)
- fix (for bug fixes)
- hotfix (for production hotfix)

Branch name format

type/short-description

Eg:

- feat/add-new-feature
- fix/app-crash
- chore/update-document
